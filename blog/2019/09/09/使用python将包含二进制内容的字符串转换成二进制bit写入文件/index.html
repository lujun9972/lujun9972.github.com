使用python将包含二进制内容的字符串转换成二进制bit写入文件

使用第三方包 bitarray,就很简单了 

import bitarray as ba
bits=ba.bitarray('0000111111')
with open('xxx.bin','wb') as f:
    bits.tofile(f)
