shell中实现字母与ASCII码之间的转换

shell中似乎没有直接的命令可以实现字母与ASCII码之间的转换。

不过通过printf我们还是可以做到这一点的。

ASCII码转字母: 

function chr() {
  [ "$1" -lt 256 ] || return 1
  printf "\\$(printf '%03o' "$1")"
}

chr 68

D

字母转ASCII码 

ord() {
  LC_CTYPE=C printf '%d' "'$1"
}

ord a

97
