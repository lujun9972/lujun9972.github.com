lshw是一个读取硬件信息的命令行工具。

安装lshw

archlinux默认并没有安装 lshw,你需要手工安装 

sudo pacman -S lshw

使用lshw

注意： 由于lshw需要读取硬件信息，因此需要使用root权限

让lshw显示出敏感信息

默认情况下，lshw会隐藏IP地址、serial number之类的敏感信息，若你希望把这类信息也显示出来，则可以执行 

sudo lshw -sanitize

获取帮助

lshw -help

会在stderr中显示简单的参数说明：

  -enable TEST    enable a test (like pci, isapnp, cpuid, etc. )
